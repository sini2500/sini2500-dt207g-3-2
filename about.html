Om arbetet

Den här uppgiften gick ut på att bygga en fristående app som hämtar data från ett API.

Appens funktioner består av att visa, radera och uppdatera data om arbetserfarenheter, vilket skulle kunna vara bra för ett framtida CV.

Javascripts fetch används för att hämta data från ett REST-API på en annan domän. API-appen har en MongoDB-databas för lagring av poster, och Mongoose användes för att kommunicera lättare med databasen.

Front-end och API har publicerats på olika servrar, via Render och Netlify, och CORS används för säker kommunikation mellan dem. Databasen görs tillgänglig via MongoDB Atlas.

Mycket av koden är densamma som föregående uppgift, men ändringar gjordes för att ansluta till den nya databasen, CRUD-operationer via routes och för att hantera datatyper som DATE och ObjectID.
