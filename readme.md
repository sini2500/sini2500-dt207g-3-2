## DT207G - backend-baserad webbutveckling

Det här är ett projekt för kursen backend-baserad webbutveckling, VT 2026.

API publicerat på Render: https://sini2500-dt207g-2-1.onrender.com

Front-end publicerad på Netlify: https://dt207-2-2-sini2500.netlify.app

---

För den här uppgiften skulle vi skapa ett litet API för att hantera arbetserfarenheter, för ett CV, i en databas.

I detta repo finns front-end delen av uppgiften, med enkel html, css och all JS som behövs i en fil.

Användaren kan :

- Visa alla poster
- Lägga till nya poster
- Uppdatera befintliga poster
- Radera poster

Formulärdata valideras med JavaScript innan data skickas till API:et, så alla fält måste finnas med, och den dubbelkollas igen i API-appen.

Main.js har en funktion för varje del av CRUD-arbetsflödet, dessa körs när en sida laddas eller knapp klickas.
